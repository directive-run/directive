How It Works (Directive Flow)

  • Facts – theme choice, locale, sidebarOpen, systemPrefersDark, translations, breakpoint (two modules: preferences + layout)
  • Derivations – effectiveTheme (resolves “system” to dark/light), isRTL (auto-tracked from locale)
  • Events – setTheme, setLocale, toggleSidebar, setSystemPreference, setBreakpoint
  • Effects – applyTheme sets data-theme on documentElement whenever theme/systemPrefersDark changes
  • Plugin – persistencePlugin persists theme, locale, and sidebarOpen to localStorage

Two modules compose into one system. Derivations auto-track facts across modules. Persistence plugin selectively saves user preferences.
