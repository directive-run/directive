How It Works (Directive Flow)

  • 4-Stage Pipeline – Extract, Summarize, Classify, Archive with simulated token costs
  • Checkpointing – Save/restore/delete state at any pipeline stage using InMemoryCheckpointStore
  • Retry with Backoff – Exponential backoff (500ms base, 2x multiplier) with configurable max retries
  • Stage Failure Injection – Force any stage to fail, observe retry behavior and pipeline halt
