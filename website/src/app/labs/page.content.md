## Theme Customization

Labs includes a theme customization panel where you can adjust the visual appearance of the Directive documentation site.

## A/B Experiments

Toggle experimental features on and off. The experiments panel lets you try out features that are still in development.

## Constraint Flow Visualization

Watch the Directive constraint engine in action. The constraint flow visualization shows how experiments trigger constraints, generate requirements, and resolve in real time -- a live demo of the runtime's reconciliation loop.
