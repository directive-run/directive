# Community Theme Vote

Every month, the Directive community votes on the default docs theme. Your choice of colors and typography becomes the first impression for every new visitor.

## How It Works

1. **Browse presets** using the palette icon in the header
2. **Pick your combo** – any of the 17 color presets and 9 typography options
3. **Cast your vote** via the share buttons (X, Bluesky, or clipboard)
4. **Most-voted wins** – the top pick becomes the site default on the 1st of next month

## Current Default

The current default theme is set via environment variables and updated monthly based on community votes. Open the Brand Preset Switcher in the header to see your current selection and all available options.

## Rules

- One vote per visitor per month (tracked client-side)
- All presets are eligible, including amber-accent ones excluded from random assignment
- Ties are broken by the Directive maintainers
- Results are announced on the first of each month via [@directive_run](https://twitter.com/directive_run)
