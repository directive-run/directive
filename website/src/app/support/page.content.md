## How to Support

### GitHub Sponsors

Become a backer -- cancel anytime. This is the primary way to support the project. Visit https://github.com/sponsors/directive-run

### Buy Me a Coffee

Fuel a late-night coding session. One-time or recurring support at https://buymeacoffee.com/sizls

### Ko-fi

Drop a tip, make our day. Support at https://ko-fi.com/sizls

### Star on GitHub

Free -- costs nothing, means everything. Star the project at https://github.com/directive-run/directive

Larger contributors get priority placement on the support page and the GitHub README. Sponsor tiers include Gold, Silver, and Bronze.
