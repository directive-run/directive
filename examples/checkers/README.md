# Checkers

A full checkers game with optional AI opponent, built as a multi-module Directive system.

## Features

- Complete checkers rules with king promotion and multi-jump
- AI mode using the `@directive-run/ai` adapter (Claude API)
- Multi-module architecture (game + chat modules)
- Time-travel debugging with 200-snapshot history

## Run

```bash
pnpm install
pnpm dev
```
