# Counter (React)

The counter example using `@directive-run/react` hooks -- `useFact`, `useDerived`, and `useEvents`.

## Features

- `useFact` for reactive count display
- `useDerived` for the doubled derivation
- `useEvents` for increment/decrement/reset actions
- Same constraint-driven clamping as the vanilla counter

## Run

```bash
pnpm install
pnpm dev
```
