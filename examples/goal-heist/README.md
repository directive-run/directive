# Goal Heist

A heist-themed AI agent orchestrator where crew members (agents) collaborate to complete mission steps.

## Features

- Multi-agent orchestration using `@directive-run/ai`
- Step-by-step or auto-run execution with satisfaction scoring
- Strategy selection and per-agent failure simulation
- Visual mission board with crew status and fact tracking

## Run

```bash
pnpm install
pnpm dev
```
