# Counter (Vue)

The counter example using `@directive-run/vue` -- `useFact`, `useDerived`, and `useEvents` as Vue refs.

## Features

- `useFact` returns a Vue ref for reactive count display
- `useDerived` for the doubled derivation
- `useEvents` for increment/decrement/reset actions
- Same constraint-driven clamping as the vanilla counter

## Run

```bash
pnpm install
pnpm dev
```
