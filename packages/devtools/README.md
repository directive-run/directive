# @directive-run/devtools

> Proprietary devtools panel for Directive systems.

This package is **not open source**. It is licensed under a proprietary license (UNLICENSED) and is not covered by the project's MIT/Apache-2.0 dual license.

The devtools panel provides visual debugging for Directive systems: dependency graphs, timeline views, state inspection, and AI agent orchestration monitoring.

For the open-source Directive runtime, see [`@directive-run/core`](https://www.npmjs.com/package/@directive-run/core).
